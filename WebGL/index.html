BarBack
ABOUT BarBack
BarBack is a fast-paced bar management game where you control a barback, restocking supplies, serving drinks, and keeping the bar running smoothly.
The game features dynamic 2D visuals, a custom shader-driven look, and a focus on quick decision-making and multitasking.

> Web demo not working? Try downloading the Windows copy! (or Linux )
Check out the web demo! https://plyr4.github.io/barback/
Built in Unity 2022.3.25f1 for Low Rez Jam 2025.
Music: Sonic Pi | Art: Aseprite | Animation: DOTween
CONTROLS
  • WASD/ARROW keys to control your character
  • WASD/ARROW keys to pick things up
  • WASD/ARROW keys to put things down
KNOWN ISSUES
  • Mini lava lamp sometimes not receiving light
